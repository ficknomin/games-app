# NextJS Game Tracker

A small app built with Next.js 16 + App Router.
It fetches data from the RAWG API and allows browsing, searching, and filtering games.

## Tech Stack

* Next.js (App Router)
* TypeScript
* Tailwind CSS
* shadcn/ui
* TanStack Query
* React Hook Form
* Zustand

## Features

* Browse games with pagination
* Search games by name
* Filter by genre and platform
* Game detail pages
* Favorite games 

## Getting Started

1. Clone the repository

2. Install dependencies

3. Create an `.env.local` file in the root:

```env
NEXT_PUBLIC_RAWG_API_KEY=your_api_key_here
```

4. Run the development server

```bash
npm run dev
```

The development server will run on http://localhost:3000
